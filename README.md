# GtaV121VatG's Web Vulnerability Scanner

## Description
This project is a simple web vulnerability scanner I designed to identify common security issues on websites.

The project has two main parts:
- A Python backend that handles the scanning logic
- A JavaScript frontend that provides a web-based interface


## Project Structure
backend/    Python backend  
frontend/   Frontend web interface  
start.bat   A One-click script to start the project on Windows  

## Requirements
- Windows computer
- Python 3 installed
- Node.js and npm installed

## How to Run the Project

### Step 1: Download
1. Go to the GitHub repository
2. Click the green Code button
3. Select Download ZIP
4. Extract the ZIP file

### Step 2: Start the application
1. Open the extracted project folder
2. Double-click start.bat
3. Two command windows will open:
   - One runs the backend server
   - One runs the frontend and opens the website in your browser

Keep both windows open while using the application.

## Usage
Enter a website URL in the frontend interface.  
The backend will analyze the site and return the results to the frontend.

## Notes
- This project does not run directly on GitHub
- The project must be run locally on a computer
- Windows only
