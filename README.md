A tool that scans website URLs to identify common security vulnerabilities and potential risks
Download the project:  
Click the green Code button   
Then Download ZIP  
Extract the ZIP file  
Run the project  
Open the project folder  
Double-click start.bat  
Two command windows will open:  
Backend (starts the server)  
Frontend (opens in your browser)  
Now just:  
Enter a website to scan in the frontend  
Backend will process the scan and show results in the frontend
